# Save File Control

This tool is designed to help you organize and back up your game save files. It is particularly useful for titles from developers like FromSoftware.

> For any issues or inquiries regarding the software, please contact me at f39597054@gmail.com.

## Features
 - backup all save files _(only supported games)_ at once
 - delete save files
 - load and unload save files _(for 10 and more saved games)_
 - seamless coop support

## Supported Games
 - Eldenring **(currently not supported)**
 - Dark Souls 1 - 3 **(currently not supported)**
 - Dark Souls Remastered **(currently not supported)**

> The development of support for these games is still in progress. If you want a specific game to be prioritized, please contact me (f39597054@gmail.com).
